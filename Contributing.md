# 贡献指南

---

## 核心主旨

> 本项目为 ArtNh 的个人数字化 portfolio/资产库。
> 
> 若有改进建议，请通过 Issues 提交理性技术探讨。

---

## 报告问题

在提交 Issue 时，请遵循以下原则：

- 请在标题中简明扼要地描述问题。
- 在描述中详细说明发现问题的环境、步骤和期望的结果。
- 请尽可能提供最小重现代码或日志截图，以便于快速定位。

---

## 提交请求

在准备提交 Pull Request 时，请确保：

- 修改或新增的代码遵循项目的整体设计美学与架构逻辑。
- 所有的代码提交信息应具备详实且客观的说明，并带有时戳标记。
- 充分验证代码在开发环境下的正常运行，确保不引入新的不可控错误。

---

我们期望在理性的技术探讨中获得共同进步，感谢您的尊重与支持。
