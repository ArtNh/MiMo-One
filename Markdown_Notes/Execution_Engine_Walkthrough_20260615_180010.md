## 变更概述
本次迭代成功在多智能体协作中枢 MiMo One 中实现了异步任务模拟执行器与三栏的数据联动。主要变更如下：
- **状态库增强**：重构了 `useAppStore.ts` 中的 `addTask` 方法，允许在挂载任务时自动生成或指定唯一任务 ID，并将 ID 返回，打破了原有的 `void` 限制，以便执行引擎对其进行后续的追踪和更新。
- **引入任务模拟引擎**：新建 `src/lib/taskRunner.ts`，定义 `TaskRunner` 类，实现 `runTask` 异步进度计算和流式日志更新模型（0% 至 100%）。
- **上下文隔离与状态联动**：
  - 重构 `App.tsx` 中的 `messages` 气泡流为按 `agentId` 映射的对象，切换智能体时载入其对应的独立历史记录，确保状态完全隔离一致。
  - 在 B 栏 Header 处增加已连接智能体状态呼吸灯：`已连接: [智能体名称]`。
  - 对 B 栏输入框的发送内容进行关键词解析（“编译/构建/生成” 派发至 Coder 编译；“分析/检索/搜索/定位” 派发至 Explorer 检索），并触发 `TaskRunner.runTask` 机制。
